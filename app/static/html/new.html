Here's how it works:

You create a set of 'outcomes' -- the possible results of the quiz.

You create a series of questions and each of those questions has a set of answers. If a user picks a given answer, that answer add a 'point' to one of the outcomes you created.

At the end of the quiz, whichever outcome has the most 'points' will be the outcome for that user. You create the outcomes, the questions, the answers, and decide which outcome a given answer adds a 'point' to.
